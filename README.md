### Installation
#### Clone code:
````
https://github.com/18Dev/fabbi_test.git
or ssh key
git@github.com:18Dev/fabbi_test.git
````

#### Run docker:
````
cd fabbi_test
docker-compose up -d
````
#### Run npm:
````
docker exec -it node bash
npm run dev -- --host
````
